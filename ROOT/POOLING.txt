Pooling Changes
-----------------------------------------
These are the things changed in the project
to add pooling.


1) Adding a context.xml file to the META-INF folder.
2) <resource-ref> tag was added to the web.xml in the
    WEB-INF folder.
3) WEB-INF/sources/gamesite/utils/DBConnection.java was
    changed to return a Connection from a DataSource

Master Slave Pooling
-----------------------------------------
These are the things changed in the project
to add pooling for multiple instances.
1) Change the localhost in META-INF/context.xml to
    the ip address of the master.
2) Change the WEB-INF/sources/gamesite/utils/DBConnection.java
    to also be able to return a connection intended for read only.
